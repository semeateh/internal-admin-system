# AGENTS.md

## 项目背景

这是一个公司内部管理系统，主要技术栈为：

- 前端：Vue + Element UI
- 后端：Python + Flask
- 数据库：MySQL
- 当前重点开发模块：流程管理功能

系统主要用于公司内部业务流转、流程审批、状态跟踪、数据管理、权限控制和操作记录。

Codex 在处理本项目时，必须优先理解业务流程和前后端调用链，而不是只针对单个文件做孤立修改。

5. 更新业务表状态

- 异常时是否 rollback
